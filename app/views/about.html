What is UnST?

Universal Storage Tool (UnST) is a project aiming at reuniting various cloud storage platforms into a single platform, allowing the users to store big data files across multiple platforms.

Yup! You've guessed it! That platform is UnST.

Open-source by design

Universal Storage Tool (UnST) is backed by the MIT open-source license.

The code is freely available for users on Github.
