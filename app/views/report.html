1. Abstract

Universal Tool Storage is a web application that gives all its users the possibility to store complex files of a large size thru another cloud services such as:Box, DropBox, Google Drive, etc. This document describes how Universal Tool Storage works and should be used.

2. Introduction

The current digital age is marked by our reliance on the cloud. Multiple applications are used in a large diversity that defines the ides of cloud storage.

A conducted study shows how "more than half of people use at least 3 different storage services". "Almost all people believe that their data habits have changed due to personal Cloud Storage. A majority of them are now using the cloud as their primary data storage. More than 50% of surveyed use a free or paid limited storage plan".

Universal Tool Storage is the application that helps users managing multiple cloud storage services from a single place. The application allows all its users to upload, download and archive files. Even if a file is lage and can not be uploaded due to its size, Universal Tool Storage will offer the possibility to fragment this file in smaller pieces and upload them in different cloud storage such that they will fit. Of course, if the user wants to download it, will be reconstructed to its initial form and delivered to him.
